\begin{frame}{Задачи лабораторной работы}
  \begin{itemize}
    \item Реализовать линейный CG, нелинейный CG, Hessian-Free Newton и L-BFGS.
    \item Реализовать \texttt{hess\_vec} для Logistic Regression + $L_2$ и Poisson Regression + $L_2$.

    \item Провести обязательные эксперименты 2.2--2.6.
    \item Выполнить трек 4: Cautious L-BFGS и отказ от Wolfe search.
  \end{itemize}

\begin{frame}{Трек 4: Cautious L-BFGS}
  \[
    \frac{\langle s_k,y_k\rangle}{\|s_k\|^2} > \varepsilon\|\nabla f(x_k)\|^\alpha,
    \qquad \varepsilon=10^{-4},\ \alpha=1.
  \]
  \begin{itemize}
    \item Wolfe search заменяется дешевым Armijo backtracking.
    \item Пара $(s_k,y_k)$ добавляется в историю только при достаточной кривизне.
    \item Если условие не выполнено, шаг принимается, но история не обновляется.
  \end{itemize}
\end{frame}

\begin{frame}{Итоговые выводы}
  {\small\begin{itemize}
    \item CG сильно превосходит GD на плохо обусловленных квадратичных задачах.
    \item Для L-BFGS оптимальный размер истории умеренный: после плато рост $L$ почти не помогает.
    \item HFN хорош на больших задачах, где полный гессиан строить нельзя, но его цена зависит от числа вызовов \texttt{hess\_vec}.
    \item Высокая оптимизационная точность не гарантирует улучшения качества на тесте.
    \item Cautious L-BFGS может заменить Wolfe на Armijo и выиграть по времени, если осторожное условие не отбрасывает почти всю историю.
  \end{itemize}}
\end{frame}
